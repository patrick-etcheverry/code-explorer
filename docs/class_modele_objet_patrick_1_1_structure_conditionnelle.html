Constructeur de la classe StructureConditionnelle.

Exemple de récupération d'une Structure Conditionnelle : str(list(p.lesStructuresConditionelles)[0])

Avec :

  • p = Objet Programme
  • list() = Conversion de "lesStructuresConditionelles" en liste afin de pouvoir sélectionner la Structure Conditionnelle que l'on souhaite
  • [0] = Première Structure Conditionnelle du programme
  • str() = Fonction permettant d'afficher le contenu d'un objet Tree-Sitter

    Résultat potentiel : (nombreDeNotes > 0)
    Paramètres
    lenodeTreeSitter: Correspond à un Node de Tree-Sitter
    progObjetPatrick: Objet instancié de la classe "Programme"
