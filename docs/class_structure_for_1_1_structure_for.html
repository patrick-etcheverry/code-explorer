def StructureFor.StructureFor.estCroissante (   self)

Retourne "true" si la structure a une incrémentation croissante et "false" sinon.

Exemple d'utilisation : p.getStructuresFor[0].estCroissante()
Avec :

  • p = Objet Programme
  • [0] = Première Structure For du programme

    Résultat potentiel : True
